# What's so2pro?
卒プロの提出用コード

# Usage
- Clone this repository.
- Run main script.
- You can see accuracy of classifier that we made.

```
$ git clone git://github.com/saxsir/so2pro.git
$ cd so2pro
$ ./bin/main examples/config.json
```

# Requirement
- PhantomJS
- Node.js
- Sqlite3
